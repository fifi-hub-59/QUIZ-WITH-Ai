# Changelog

## [1.0.0] - 2026-07-22
### Added
- Complete UI overhaul to Jetpack Compose Material 3 with a Cosmic Dark Theme.
- Animated Screen transitions and Shared Element-like effects.
- New Screens: Splash, Home, Game, Settings, Store, Profile, Leaderboard.
- Local Persistence using Room Database (Coins, XP, Avatar unlocks, Settings).
- Simulated Monetization and Rewarded Ads framework for easy integration.
- Custom fonts via Typography and Adaptive Layouts.
- Audio engine for SFX and dynamic background music.
- Comprehensive UI state management using MVVM and StateFlow.
- Removed manifest warnings and optimized Gradle configurations.
